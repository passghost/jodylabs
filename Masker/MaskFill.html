How to Use the Masking Tool

  • Select up to 3 input images (these will be used for masking).
  • Select a mask image (white or black pixels determine placement).
  • Set output file name and adjust image size and frequency using sliders.
  • Toggle masking options (white pixels, random sizes, random rotation).
  • Enable/disable hue shifts for color variation.
  • Click Process Images to generate and download the masked image.
  • Preview will appear below after processing.
