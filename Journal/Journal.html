Journal Entry: Scrolling Starfield Game

Today, we made significant progress on our scrolling starfield game, a project that combines elements of space exploration and combat. We started by creating a dynamic starfield background that scrolls smoothly, giving the impression of traveling through space. Adding the player ship with WASD controls and a firing mechanism was a fun challenge. We also implemented enemy ships that move in geometric patterns and shoot projectiles, adding an exciting layer of complexity to the game. The visual effects, including the glowing lasers and the flame effect when the player ship moves forward, really brought the game to life.

One of the most rewarding aspects of this project was seeing all the elements come together seamlessly. The health system for both the player and enemy ships added a strategic component, making the gameplay more engaging. We also tweaked various parameters, like increasing the speed of the stars and adjusting the firing rates, to enhance the overall experience. This project has been a great opportunity to apply our coding skills and creativity, and I'm excited to see how we can further develop and refine the game in the future.

Journal Entry: Dad's Web Counter

Creating "Dad's Web Counter" was a collaborative and nostalgic journey, blending modern web technologies with a classic concept. The project began with a simple HTML file that displayed numbers in rows and columns, counting to infinity. As we progressed, we added dynamic elements like a bouncing ball that interacted with the numbers, changing their colors and resetting their values. The ball's movement was refined to ensure smooth animation and proper collision detection with the browser walls and the title. We also incorporated a vibrant glow effect and a color-changing mechanism to enhance the visual appeal.

Throughout the development process, we focused on optimizing performance to ensure the simulation ran smoothly without breaking. This involved reducing the number of active timers, optimizing DOM manipulations, and using efficient counting mechanisms. The final touch was adding a nostalgic title, "Dad's Web Counter," positioned on the far left of the screen, reminiscent of the classic programs your dad used to write in BASIC. The project not only showcases technical skills but also brings back fond memories, making it a meaningful and enjoyable experience.
